# 二偶 Live Memory Archive

一个用于记录二次元偶像、女性声优、动画 / 游戏相关 Live 和线下活动的个人网页资料库。

## 已实现功能

- 演唱会资料库卡片列表
- 首页统计、最近回忆、企划分布
- 新增、编辑、删除演唱会条目
- 详情页：基本信息、感想、曲目单、购票、座位、周边、同行人、行程、记忆碎片
- 图片上传、预览、说明、删除
- 可从相册设置封面图
- 搜索、标签筛选、企划筛选
- 中文 / 日语界面切换
- 轻量 JSON 备份导出
- 完整 JSON 备份导出，包含媒体 Base64
- JSON 备份导入，可选择合并或覆盖
- IndexedDB 本地持久化
- Supabase 登录和云同步预备版
